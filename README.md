**Absolute Blooters** is a lightweight website for watching football goals as they happen.

Goal links are automatically fetched from the r/soccer subreddit and posted in near real time. Goals are grouped by match with live scores, and mirror links are provided for broken videos or alternate angles.

Perfect for quickly catching up on goals without digging through match threads.

## Tech Stack

- **Backend:** Go: handles data ingestion, processing, and API endpoints
- **Frontend:** React + Vite: fast UI with instant updates
- **Database:** PostgreSQL: persists matches and goals
- **Monitoring:** Prometheus + Grafana: metrics collection for backend performance
